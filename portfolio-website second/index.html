Website convert mp3 audio files to text.

created a website that can convert mp3 audio files to text. I used serverless technology to transcribe audio lambda and S3 provided by AWS. The process of setting up the website was simple. First, I created a new AWS lambda function and added the transcribe audio code. I also set up an Amazon S3 bucket to store the audio files, and transcribed text. Then I used an API to connect the Lambda function to my website so that when a user uploads an audio file, the conversion takes place automatically. Weather App

Weather App that brings real-time weather information to your fingertips. 📱🌤️ The main highlight of this app is its seamless integration with a Python Flask API using the HTTP protocol. The Flutter app communicates with the Flask API to retrieve weather data, fetching information from OpenWeatherMap's extensive database for processing. This collaboration allows us to present accurate weather forecasts to our users and keep them updated with the latest weather conditions wherever they are. 🌏☀️🌧️ I'd like to give a special shout-out to the Flask framework for making it effortless to build and deploy the backend for this project.
